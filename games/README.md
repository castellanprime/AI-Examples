# Othello

This is a game-playing AI for the game [Othello](https://en.wikipedia.org/wiki/Reversi). 

## Files

- NoAIOthello.py Better two-player game with no AI


### Features

* Written in Python
  - GUI is Pygame/wxPython
* Uses async
* Minimax algorithm 
  - Alpha-beta pruning

To run:
> ./NoAIOthello.py
