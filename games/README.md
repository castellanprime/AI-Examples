# Othello

This is a game-playing AI for the game [Othello](https://en.wikipedia.org/wiki/Reversi). 

## Files

- rec_othello.py Barebones two-player game with no computer and no AI


### Features

* Written in Python
  - GUI is Pygame/wxPython
* Uses async
* Minimax algorithm 
  - Alpha-beta pruning

To run:
> ./othello.py
